## Learnings
- WizardStep is a union of 10 string literals matching the wizard steps (name, race, background, class, order, stats, skills, vocation, equipment, review)
- WizardData has 16 fields including nested types from @/lib/types (Race, HumanCulture, StartingClass, Background, KnightOrder, Vocation, CharacterStats, ArmorPreset)
- StepProps is the base props type used by step components: { data: WizardData, setData: React.Dispatch<React.SetStateAction<WizardData>> }
- Some steps extend StepProps with additional props (skills, totalProficiencies, rollMoney) - these will need to be handled in extraction tasks T025-T027
