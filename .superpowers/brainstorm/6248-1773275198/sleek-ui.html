A: Almendra Everything

One body font for all non-title text. Almendra's calligraphic character carries through nav, stats, buttons. Relies on weight/size/color for hierarchy. Simplest font stack (2 fonts total). The ✦ separators add elegance.

B: Cinzel Light for UI Chrome

Cinzel at weight 400 (lightest) + small size + letter-spacing. Much sleeker than before — the Roman capitals become delicate inscriptions rather than shouting. Nav stays in Almendra with ◆ separators. 3-font stack.

C: Cormorant Garamond for UI

Elegant, high-contrast Garamond. Thin strokes give it natural sleekness. Wide weight range for subtle hierarchy. Nav uses Cormorant too with minimal · separators. 3-font stack but the most refined feel.

What changed

Navigation: No more uppercase blocks — graceful text with ornamental separators (✦ ◆ ·)
Buttons: Ghost/outline style — transparent background, thin borders, subtle hover
Stat badges: Slimmer, less padding, lower-key borders
Labels: Lighter weight, reduced letter-spacing
